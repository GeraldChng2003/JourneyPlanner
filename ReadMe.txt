Installation
------------
Note: The following application does not use Firebase, as such the server and the application would have to be locally hosted.

1. Navigate to Frontend folder
2. Run npm install to install dependencies.
3. Start the app with "npx expo start"
4. Open another terminal and navigate to the backend folder
5. Run the server.js with "node server.js"
6. Manually change the serverIP in each .txt file to match your current IP address

Usage
-----
- Open the app in a computer
- Follow on-screen instructions.
